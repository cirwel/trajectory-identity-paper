```python
def nth_fibonacci(n):
    """Return the nth Fibonacci number, with F(0)=0 and F(1)=1."""
    if n < 0:
        raise ValueError("n must be non-negative")

    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

It runs in `O(n)` time and uses constant space.
